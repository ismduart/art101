Task 1: Pseudocode an everyday task

Sending important documents in the mail.

  1. Figure out where you're mailing this to.
  2. Grab an empty, plain envelope.
  3. Rotate the envelope so you're looking at it horizontally. Hot dog style.
  4. In the top left corner, write your address. An Example:
    Sammy Slug
    1156 High St
    Santa Cruz, CA 95064
  5. In the center, write the recipient's address formatted similarly to yours.
  6. Place documents inside and seal.
  7. Place appropriate postage in the top right corner.
  8. Drop off in the nearest Post Office Box.
  9. Celebrate when the recipient gets it!

Task 2: Pseudocode a simple computer game

Creating A Rolling Dice Game

  1. Import a library that will randomly generate a number 1-6
  2. Accept user input.
  3. Establish the amount of dice being rolled.
  4. Prompt user for the amount of dice.
  5. Validate user move.
  6. Evaluate possible outcomes.
  7. Randomize numbers 1-6 per die rolled.
  8. Display results.
